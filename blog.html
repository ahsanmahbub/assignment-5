Important FAQ - DOM in JavaScript

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing developers to interact with the content, structure, and styles of a webpage dynamically through scripting languages like JavaScript.

2. How do you select an element from the DOM?

You can select an element from the DOM using methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName(). These methods allow you to access specific elements based on their IDs, classes, or selectors.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that allows you to attach a single event listener to a parent element instead of adding multiple listeners to individual child elements. The event listener listens to events triggered by child elements and determines what triggered the event using event bubbling. 4. How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using methods like setAttribute() and getAttribute(). To manipulate styles, you can access the style property of an element and assign values to CSS properties, for example, element.style.color = 'blue';.
